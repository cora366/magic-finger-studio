# Examples

이 폴더는 Magic Finger Studio의 예제 프로젝트와 샘플 코드를 저장합니다.

## 목적

학습자가 단계별로 참고할 수 있는 예제와 완성 코드를 제공합니다.

## 저장 대상

- STEP별 예제 코드
- 기능별 샘플 프로젝트
- 실습 완성본
- 데모 프로젝트
- 테스트 예제

## 규칙

- 하나의 예제는 하나의 폴더로 관리합니다.
- 예제마다 README.md를 작성합니다.
- 예제에는 HTML, 프롬프트, 설명 문서를 함께 저장합니다.
